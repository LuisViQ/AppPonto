# Documentacao AppPonto

Indice:

- ARCHITECTURE.md -> Visao geral, fluxo offline-first e sincronizacao.
- APP.md -> App mobile (telas, servicos, Realm, UI).
- API.md -> API Express, endpoints e payloads.
- DATABASE.md -> Modelo do MySQL, campos extras e triggers.
- FLOW.md -> Fluxo tecnico detalhado, funcao por funcao.
- SYNC_FLOW.md -> Fluxo detalhado de salvar e sincronizar (push/pull/Outbox).
- FILE_INDEX.md -> Cada arquivo com suas funcoes/props/exportacoes.

Como ler:

1. Comece por ARCHITECTURE.md para entender o fluxo.
2. Depois veja APP.md e API.md.
3. Use FILE_INDEX.md como referencia rapida.
